## Главная (`/`)

1. [ ] **Prelude · 4K-safe.** На ≥2560px `h1` не разрывает строку криво, не уезжает за край `.prelude-wrap`. Потолок `clamp(56px, 10vw, 140px)` работает.
2. [ ] **Prelude · stats-head.** 3 колонки (Мастерская · с 2020 / В архиве · 235 работ / Сезон · MMXXVI). Ничего не обрезается, hairline под ними корректный.
3. [ ] **Manifesto.** Drop-cap «М» читается, обтекается текстом, не наезжает на соседние абзацы. Три `tenet` стакаются в одну колонку ≤900px.
4. [ ] **Calendar.** Текущий месяц визуально `tight` (node светло-бежевый, bar ~90%). Январь–Март прошедшие (closed). Цвета `free/tight/closed` различимы.
5. [ ] **Genres · цены.** Все 8 жанров в порядке: 2 500 / 3 500 / 6 000 / 7 500 / 8 000 / 8 500 / 40 000 / 60 000 ₽. Совпадает с `REAL_DATA.md §4`, ничего не округлено.
6. [ ] **Formula · арифметика.** Клик по `Дипломная` → receipt пересобирается; + `Срочно (×1.6)` → итог `64 000 ₽`. Три дополнения все включены → `+7 500 ₽` сверху над базовой математикой.
7. [ ] **Formula · CTA.** «Оформить заявку» ведёт на `/order?type=<код>` или `/order?topic=<название>`; страница order.astro подхватывает `type` (`urlType` pills) / `topic` (`#fTopic`).
8. [ ] **Archive · фильтр.** 25 строк. Клик «ВКР и дипломы» показывает только строки с `data-category="ВКР и дипломы"`, счётчик `показано N из 235` обновляется.
9. [ ] **Archive · click-through.** Клик по строке открывает `/doc/<file>` — страница рендерится, контент на месте.
10. [ ] **Method · табы.** 5 табов (Conspectus … Actio), клик и mouseenter (≥900px) переключают активный. Стрелки ↑↓←→ меняют фокус и активный таб, зацикливаются.
11. [ ] **Testimonia · reviews-wall.** Обе полосы плавно едут в разные стороны, клик по любой карточке открывает lightbox, prev/next/close работают.
12. [ ] **Guarantees.** 4 плашки с § I–IV. На ≤1000px — 2×2, на ≤600px — 1 колонка.
13. [ ] **Q&A.** 10 `<details>`. Клик — открывает/закрывает, `+` поворачивается в `×` при open. Первый открыт по умолчанию не обязателен (сейчас все закрыты).
14. [ ] **Epilogue · каналы.** 4 кнопки (Telegram, VK, MAX, Email). Клики реально открывают нужный URL; handle совпадает с `REAL_DATA.md §5`.
15. [ ] **Footer.** Монограмма «АС» слева, 3 колонки (Салон / Форматы / Контакты), bottom-bar с «© MMXX—MMXXVI». Ссылки в «Форматы» ведут на `/order?type=...`.
16. [ ] **Scroll-progress.** При прокрутке сверху экрана едет золотая нить (1px), глубина прокрутки растёт. Тёмный и светлый темы.
17. [ ] **Curtain.** При первой загрузке `/` накрывает экран obsidian-фон с золотой «АС» по центру, уезжает вверх за ~800мс. На повторных навигациях `astro:page-load` — тот же эффект.
18. [ ] **Reveal-on-scroll.** Блоки с `data-reveal` всплывают снизу при попадании в viewport. Chapter-head каждой главы, manifesto-body, genres/archive/qa списки, epilogue-channels.
