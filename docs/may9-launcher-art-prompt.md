# Промт для картинки модалки-анонса «По рассказам»

Картинка показывается в полноэкранной модалке на главной сайта при первом заходе. Размер: **1200×750** (соотношение 16:10), возможно квадратное обрезание на мобиле.

Формат: **PNG** (с лёгкой компрессией) или **JPEG q=85**. После генерации положить в:
`astro-site/public/may9/launcher-art.png` или `.jpg`.

Затем в `May9Launcher.astro` поменять путь:
```html
<img src="/may9/launcher-art.png" ... />
```

## Технические требования к изображению

- **Размер**: основной — 1200×750 (для модалки и OG). Опционально дополнительная квадратная версия 1080×1080 для соцсетей.
- **Вес**: до **450 KB** в PNG. До **180 KB** в JPEG.
- **Профиль**: sRGB.
- **Без текста на латинице**, без узнаваемых кириллических надписей (если только не вариант C — там текст-цитата).
- **Без видеомусора**: чистая ровная композиция, без лишних деталей по краям.
- **Тёплые тона** на холодном фоне — это главный визуальный код проекта.

## Что делать после генерации

1. Сохрани файл как `launcher-art.png` (или `.jpg`).
2. Положи в `astro-site/public/may9/launcher-art.png`.
3. В `astro-site/src/components/may9/May9Launcher.astro` поменяй строку:
   ```html
   <img src="/may9/launcher-art.svg" ... />
   ```
   на:
   ```html
   <img src="/may9/launcher-art.png" ... />
   ```
4. Отдельно: если хочешь использовать ту же картинку как OG для соцсетей при шаринге `/may9/`, замени `og-may9.png` или укажи launcher-art в фронт-маттере страницы.

## Резерв на случай если картинка не успеет

Сейчас в репозитории лежит **типографический SVG-плейсхолдер**:
`astro-site/public/may9/launcher-art.svg` — стилизованное письмо
с цитатой, фотокарточкой и стаканом. Он работает по умолчанию,
пока ты не подменишь.
