# CONTRIBUTE_PLAN — редизайн `/contribute` («Поделиться»)

> Дата: 2026-04-19. Основа: аудит `contribute.astro` (1056 строк), `api/routers/contribute.py`, `CLAUDE.md`, пользовательский бриф («тотально в нашем дизайне, изучи суть, какие фичи»).

---

## 1. Зачем вообще этот раздел

Архив из **235 работ** в библиотеке — это база проекта. Владелец строил его с 2020 года через Telegram-канал «Кладовая ГИПСР» — студенты присылали свои защищённые работы, он публиковал обезличенно для следующих поколений. Раздел `/contribute` — **веб-форма этого потока**, чтобы не зависеть от переписок в TG.

**Ценность раздела:**

- Для владельца — **бесплатный контент-поток** для архива. Каждая принятая работа → +1 к каталогу → больше трафика через SEO.
- Для студента-доноpa — **сделать что-то хорошее**: его работа поможет следующим студентам в такой же ситуации. Плюс анонимность.
- Для студента-читателя — больше примеров для ориентирa.

**Ключевой инсайт из чата с владельцем:** «идея такая — вызвать доверие, плюс, по сути, я же переехал с телеграмма на сайт, апдейт — новый уровень». То есть `/contribute` — это не просто форма, а **история про сообщество**: люди отдают, другие берут, круг замкнут.

## 2. Что не так сейчас (итог аудита)

**Дизайн:**
1. Большой glow-hero с пословной анимацией — не editorial.
2. Upload-zone — серая dashed-рамка (`border: 1px dashed var(--brd)`), не в Dark Couture.
3. Форма — та же старая `.pill` и `.field` что на `/order` до редизайна.
4. `.step-no-r` с `opacity: .06` — римские цифры почти не видны.
5. Submit-кнопка — не solid-gold editorial.
6. Success-screen — legacy `.premium-success` с кнопкой-плашкой «АС» на золотом фоне.

**UX:**
7. Нет автозаполнения. Имя файла `Курсовая - Психология - Конфликты в семье.docx` содержит всю информацию, но пользователь вбивает руками.
8. Нет чипов-подсказок для «Тип работы» / «Предмет» — пользователю нужно угадать ожидаемые значения.
9. Нет явного «что вы получаете взамен» — кроме «поможет кому-то».
10. Нет социальных счётчиков («уже в архиве 235 работ», «за месяц сдано N»).
11. Нет гарантии анонимности явно.
12. Нет гайдлайнов: какие работы подходят, какие нет.

**Контент:**
13. Текст обезличен — но не объясняет «как мы обезличиваем» (ФИО из текста? ВУЗ? научрук?).
14. «Контакт автора виден только модератору» — сказано, но не выделено как гарантия.
15. Раздел воспринимается как «ещё одна форма», а не как часть истории проекта.

## 3. Принципы редизайна

- **Dark Couture, как /order и главная.** Cormorant Garamond, Inter Tight, JetBrains Mono. Editorial chapter-head с римской цифрой.
- **Скромный тон.** «Поделитесь работой — сделайте архив чуть больше.» Без «Присоединяйтесь к сообществу!» и прочей корпоративщины.
- **Анонимность явно.** Отдельная плашка «Что видит архив / что остаётся у нас» с двумя колонками.
- **Умный upload.** По имени файла автоматически распознаём тип и предмет, подставляем.
- **Чипы вместо input'ов** для «Тип работы» и «Предмет» — быстрее и унифицировано с архивом.
- **Социальные счётчики** — ненавязчиво в hero.
- **Success — как благодарность**, не как техническое подтверждение.

## 7. Фича: автозаполнение из имени файла

Типичные имена файлов от студентов:
- `Курсовая - Психология - Конфликты в семье.docx`
- `ВКР_Петрова_Социальная работа_2026.docx`
- `Реферат по философии - Кант.docx`
- `Отчёт по практике - школа №12.pdf`

**Алгоритм на клиенте (чистый JS, без Codex):**

1. Убрать расширение.
2. Разбить по `[-_,]|\s+`.
3. Первый токен → проверить в словаре типов работ (курсовая/реферат/вкр/диплом/эссе/отчёт/конспект/магистерская). Если совпало → category.
4. Следующий токен → проверить в словаре предметов (psychology → Психология, social → Социальная работа, ...). Если совпало → subject.
5. Оставшиеся токены → title.
6. Убрать из имени ФИО (regex: 3 слова с большой буквы подряд) — показать пользователю как «мы убрали из имени» для transparency.
7. Если не распозналось — оставить поля пустыми, показать чипы.

Всё это — pure JS на клиенте.

## 10. Что НЕ делаем

- Регистрацию / личный кабинет для доноров (не нужен, overkill).
- Публичный счётчик «вы — N-й донор за месяц» (вульгарно).
- Бейджи «топ-донор» на карточках работ (нарушает анонимность).
- Сравнение с TG-каналом «смотрите, у нас лучше» — чат закрыт этикой.
- Автопубликацию без модерации.
- Откат публикации по запросу автора через форму (это через модерацию вручную).
